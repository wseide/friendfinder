# [FriendFinder]( https://hidden-savannah-42359.herokuapp.com/)
## Overview
Friend Finder application. Take a survery to determine who are you compatible with.
## Dependencies
* body-parser
* express
* path
## Installation
### Install Locally
```
git (https://github.com/wseide/friendfinder)
npm install
```
### Command
```
node server.js
```
